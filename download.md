
# Download

##### libdill

```
$ wget http://libdill.org/libdill-0.7-beta.tar.gz
$ tar -xzf libdill-0.7-beta.tar.gz 
$ cd libdill-0.7-beta
$ ./configure
$ make
$ sudo make install
```
